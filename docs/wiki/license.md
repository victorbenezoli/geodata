## IBGE data

The geospatial data returned by this package is obtained from the
[IBGE Mesh API](https://servicodados.ibge.gov.br/api/v3/malhas) and the
[IBGE Localities API](https://servicodados.ibge.gov.br/api/v1/localidades).

Use of this data is subject to the terms established by IBGE.
See the [IBGE portal](https://www.ibge.gov.br) for more information.
